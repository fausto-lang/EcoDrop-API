# Guia

## PASO 1

Cuando hayan clonado el repositorio, se van a abrir 2 terminales, uno es para el backend y el otro es para el frontend y se mueven en cada terminal a esa carpeta

TERMINAL 1

```
    cd frontend
```

TERMINAL 2

```
    cd backend
```

## PASO 2

En su terminal del frontend deberian primero descargar los paquetes necesarios con el siguiente comando:

```
    pnpm install
```

## PASO 3

Para hacer correr el frontend lo hacen con el siguiente comando:

```
    pnpm run dev
```

les dara un enlace para ver la pagina.
